Detailed Description

Klasa bazowa dla wszystkich obiektów statycznych w silniku.

Obiekty posiadają po dwie zmienne na orientację i pozycję. Przechowywana jest zawsze pozycja z poprzedniej klatki, dzięki czemu można interpolować położenie. (Wyświetlanie jest opóźnione w czasie o jedną klatkę, żeby interpolacja nie musiała wyprzedzać faktycznych położeń).

Swapowanie następuję w funkcji DynamicObject::move.
