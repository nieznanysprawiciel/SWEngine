Detailed Description

Klasa przechowuje metadane na temat typów actorów silnikowych.

Klasa może byc użyta nie tylko do wydobywania metainformacji o typie, ale również może posłużyć do pobrania wartości dla konkretnego obiektu. Przechowywany wewnętrznie wskaźnik m_actorPtr może zostać ustawiony przy pomocy funkcji reset. Skutkuje to zaktualizowaniem wskaźnika w całej hierarchii.

Wskaźnik może być nullptrem, wtedy zwracane są jakieś domyślne wartości dla parametrów.
