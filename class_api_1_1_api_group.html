Detailed Description

Klasa bazowa dla klas w EngineInterface służących do grupowania funkcji interfejsu dla użytkownika (klasy GamePlay i obiekty aktorów).

Klasa posiada statyczny wskaźnik na Engine i EngineContext. Dzięki temu nie trzeba go przekazywać do każdej klasy z osobna. Dodatkowo można w tej klasie umieszczać wskaźniki na inne obiekty, które wymagają bezpośredniego dostępu.
