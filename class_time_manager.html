Detailed Description

Klasa obsługuje wszystko co związane z czasem.

Do zadań klasy należy:

  • pobieranie i przeliczanie czasu
  • obsługa eventów czasowych
  • Robienie statystyk wykonania poszczególnych fragmentów kodu

Funkcje zaczynające się od query pobierają czas z systemu. Funkcje get zwracają wartości obliczone wcześniej i przechowywane w klasie. Tak naprawdę powinno się raczej korzystać z funkcji get, ponieważ w silniku nie liczy się czas aktualny, ale czas jaki jest w danej klatce.

int64 TimeManager::QueryCurrentTime ( )

Wywołuje QueryPerformanceCounter, żeby pobrać aktualny czas.

Należy pamiętać, że najczęściej interesuje nas czas aktualnej klatki albo lag. Można je pobrać funkcjami TimeManager::GetTime i TimeManager::GetFrameLag.

Ta funkcja pobiera aktualny czas nieżny od wyświetlania klatek.
