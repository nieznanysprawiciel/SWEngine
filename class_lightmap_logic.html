Detailed Description

Klasa obsługująca generowanie lightmap.

Aby włączyć generowanie należy wcisnąć przycisk 1, 2, 3, 4 lub 5. Wybierana jest jedna z funkcji do generowania.

Wywołuje generowanie lightmap w silniku.

Funkcja pobiera obiekty statyczne ze sceny. Ponieważ mechanizm eventów w silniku nie pozwala na razie na podczepianie wielu funkcji obsługi pod jeden event, to wywoływana jest też funkcja LoadScene, jeżeli wciśnięto przycisk L.
