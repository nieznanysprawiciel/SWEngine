58 Version history:
59 
60 1.4a February 22nd, 2013
61 Removed unnecessary and counter-intuitive solar radius parameters
62 from the interface of the colourspace sky dome initialisation functions.
63 
64 1.4 February 11th, 2013
65 Fixed a bug which caused the relative brightness of the solar disc
66 and the sky dome to be off by a factor of about 6. The sun was too
67 bright: this affected both normal and alien sun scenarios. The
68 coefficients of the solar radiance function were changed to fix this.
69 
70 1.3 January 21st, 2013 (not released to the public)
71 Added support for solar discs that are not exactly the same size as
72 the terrestrial sun. Also added support for suns with a different
73 emission spectrum ("Alien World" functionality).
74 
75 1.2a December 18th, 2012
76 Fixed a mistake and some inaccuracies in the solar radiance function
77 explanations found in ArHosekSkyModel.h. The actual source code is
78 unchanged compared to version 1.2.
79 
80 1.2 December 17th, 2012
81 Native RGB data and a solar radiance function that matches the turbidity
82 conditions were added.
83 
84 1.1 September 2012
85 The coefficients of the spectral model are now scaled so that the output
86 is given in physical units: W / (m^-2 * sr * nm). Also, the output of the
87 XYZ model is now no longer scaled to the range [0...1]. Instead, it is
88 the result of a simple conversion from spectral data via the CIE 2 degree
89 standard observer matching functions. Therefore, after multiplication
90 with 683 lm / W, the Y channel now corresponds to luminance in lm.

92 1.0 May 11th, 2012
