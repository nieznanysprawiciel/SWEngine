void ActorBase::event ( Event *  new_event)
inlineprotected

Funkcja pozwala wysłać event, który będzie potem przetworzony przez klase FableEngine. Eventy są metodą komunikacji pomiedzy silnikiem graficznym, silnikiem fizycznym, AI i silnikiem kolizji, a modułem silnika odpowiedzialnym za fabułę. Istnieje szereg eventów wbudowanych, wysyłanych przez silnik, można również definiować własne nowe eventy poprzez dziedziczenie z klasy Event. Event może być wysłany przez dowolny objekt poprzez wywołanie funkcji ActorBase::event. Aby wysłać własny event trzeba przeciążyć jedną z funkcji klas wbudowanych, która jest potem wywoływana przez silnik i wywołać tę funkję.

Za zwolnienie pamięci po klasie Event odpowiada klasa FabelEngine (jest to robione automatycznie po wywołaniu funkcji obsługi, użytkownik nie musi się tym przejmowac).
