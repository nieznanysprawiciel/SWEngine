Detailed Description

Bufor stałych dla shadera zmieniający się dla każdej części mesha.

Obiekt zawiera macierz przekształcenia świata dla danej części obiektu oraz dane o materiałach. Materiały są wektorami 4-wymiarowymi tylko ze względu na to, że takie są rejestry na GPU. Używane są tylko pierwsze 3 elementy wektora. Jedynie w kanale Diffuse ostatnia współrzędna odpowiada za przezroczystość.
