Detailed Description

Bufor stałych dla shadera zmieniający się raz na klatkę.

Bufory stałych dla shaderów muszą mieć wielkość będącą wielokrotnością 16. Wszystkie dane w GPU są przechowywane w postaci wektorów 4-wymiarowych. Dlatego lepiej nie wysyłać danych o innej liczbie wymiarów, bo na pewno coś się zepsuje.

Ewentualnie jeżeli chcemy usprawnić dostęp do danych po strnie CPU i podawać wektory 3-wymiarowe, to trzeba uzupełnić czwarta współrzędną jakimś pustym floatem.
