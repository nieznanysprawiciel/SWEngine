Detailed Description

Klasa przechowująca tekstury.

Istnieją obiekty nazywające się ID3D11Texture2D i są teksturami, ale stworzenie takiego nie wystarczy, żeby shader mógł go użyć. Do tego trzeba stworzyć widok i właśnie jego przechowuje ta funkcja.

TextureObject * TextureObject::create_from_file ( const std::wstring &  file_name)
static

Tworzy z podanego pliku obiekt tekstury o identyfikatorze z pola id.

Generalnie ta funkcja nadaje się do używania wielowątkowego. Obiekt directXa ID3D11Device jest synchronizowany wewnętrznie i nie trzeba się o to martwić. Z tego względu można tworzyć wiele różnych zasobów jednocześnie.

W przeciwieństwie do ID3D11Device, ID3D11DeviceContext nie jest synchronizowany, więc nie można renderować z wielu wątków jednocześnie przy użyciu jednego obiektu.
