Detailed Description

Obiekt opakowujący bufor DirectXa.

Bufor wierzchołków i bufor używają tego samego interfejsu ID3D11Buffer, dlatego nie ma potrzeby tworzenia oddzielnych klas. Obiekty tego typu mogą także służyć do przekazywania parametrów shaderom, jako bufory stałych.
