void Api::RenderingApi::RenderOnce ( RenderPass *  pass)

Dorzuca do kolejki renderingu nowy przebieg, który zostanie wyrenderowany tylko raz.

Funkcja jest przydatna, jeżeli chcemy wyrenderować teksturę, która będzie niezmienna przez całą grę.

Po wyrenderowaniu wysyłany jest event RenderOnceEndedEvent. Nie ma gwarancji, że renderowanie odbędzie się w tej samej klatce, w której zostało wysłane żądanie. Jeżeli pętla renderująca będzie za bardzo obciążona, to z kolejki będzie wybieranych tylko kilka żądań, a pozostałe zostaną odłożone do kolejnych klatek.

Attention
W obecnej wersji event musi zostac odebrany, żeby zwolnić obiekt RenderPass i wszystkie zasoby, które zostaną w nim umieszczone. W przyszłości trzeba zrobić jakiś mądrzejszy mechanizm.
